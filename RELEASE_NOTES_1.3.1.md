# SystemCare 1.3.1

A small fix release.

## 🐞 Fix

- **Startup splash now shows the new logo.** The native splash image displayed by the app host before
  the UI loads still had the previous checkmark logo baked in, so the new neon-shield logo only
  appeared once the animated splash took over. The splash now uses the new logo from the first frame.

## 📥 Install

Download **`SystemCare-Setup.exe`** below and run it (administrator rights required). Upgrades install
in place. The installer is unsigned, so SmartScreen may warn - choose **More info > Run anyway**.
