# SystemCare 1.4.0

A visual-polish release that makes the Dashboard feel more refined, with smoother graphics and subtle new
animations - all in the existing cyberpunk "Night City" style.

## ✨ Dashboard refinements

- **Health gauge** - a glass gradient track, faint precision tick marks, a glowing dot at the leading tip
  of the score sweep, and a brief glow flash when a fresh score lands.
- **Flowing sparklines** - the CPU / RAM / network graphs are now smooth curves with a glowing dot
  tracking the live value, instead of straight-segment lines.
- **Counting numbers** - the big CPU% and Memory readouts count up/down smoothly instead of snapping each
  second.
- **Smarter drive cards** - usage bars glide to their value and shift colour (cyan -> yellow -> magenta)
  as a drive fills up, with a subtle hover lift.
- **Staggered entrances** - dashboard sections and tiles cascade in gently when you open the page.

Everything stays lightweight (short, eased, GPU-friendly motion), so responsiveness is unchanged.

## 📥 Install

Download **`SystemCare-Setup.exe`** below and run it (administrator rights required). Upgrades install in
place. The installer is unsigned, so SmartScreen may warn - choose **More info > Run anyway**.
